# KVGrav

Библиотека эмуляции гравитации.

Текущие возможности:

- взаимодействие каждого обьекта друг с другом
- взаимодействие каждого обьекта с внешним полем
- настройка параметров (скорость, радиус и сила внешнего поля)

В разработке:

- Энерция

## Установка

Yarn

```shell
yarn add kvgrav
```

npm

```shell
npm install kvgrav
```
